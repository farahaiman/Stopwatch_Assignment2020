# ⏱️ Stopwatch Application

A simple and responsive stopwatch application built using HTML, CSS, and JavaScript.

## 🔗 Live Demo

https://farahaiman.github.io/Stopwatch_Assignment2020/

## 📖 Project Overview

This project is a basic stopwatch that allows users to start and reset the timer. It was created to practice JavaScript functionality and user interface development.

## ✨ Features

- Start Timer
- Reset Timer
- Simple User Interface
- Lightweight Design
- Responsive Layout

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript


## 🚀 How to Use

1. Click the **Start** button to begin the timer.
2. Click the **Reset** button to reset the timer to zero.
